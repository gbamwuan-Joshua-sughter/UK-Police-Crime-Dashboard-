#UK_Police_Crime_Analysis_Dashboard (Microsoft Excel)

Project Overview

This project presents an interactive crime analytics dashboard developed in Microsoft Excel using the UK Police Recorded Crime dataset. The analysis explores crime patterns across police forces, offence groups, financial years, and reporting quarters to uncover trends that can support public safety planning and resource allocation.

The project demonstrates the complete data analysis workflow, from data preparation and exploratory data analysis (EDA) to dashboard design and business insight generation.

---

Business Problem

Police agencies collect large volumes of crime data every year, making it difficult to identify meaningful trends without effective analysis and visualization.

This project transforms raw crime records into an interactive dashboard that enables stakeholders to monitor crime levels, compare police force performance, identify high-risk crime categories, and analyze changes in crime over time.

---

Objectives

- Analyze crime trends across financial years.
- Compare recorded offences across UK police forces.
- Identify the most common offence groups and subgroups.
- Examine quarterly crime patterns.
- Build an interactive dashboard for data-driven decision-making.

---

Dataset Overview

The dataset contains over 250,000 crime records collected from UK Police forces and includes:

- Financial Year
- Financial Quarter
- Police Force Name
- Offence Description
- Offence Group
- Offence Subgroup
- Offence Code
- Number of Offences

Data Preparation

The following data preparation steps were completed:

- Consolidated multiple yearly datasets into a single master dataset.
- Standardized column names and formats.
- Removed duplicate and unnecessary records.
- Checked for missing values and inconsistencies.
- Prepared the data for Pivot Table analysis.

---

Exploratory Data Analysis (EDA)

The analysis focused on answering questions such as:

- Which police forces recorded the highest number of crimes?
- Which offence groups are most common?
- How have crime levels changed over time?
- Are there seasonal or quarterly crime trends?
- Which offence subgroups contribute most to total crime?

---

Dashboard Features

The interactive dashboard includes:

- Total Recorded Offences (KPI)
- Number of Police Forces
- Crime Trend by Financial Year
- Crime Trend by Quarter
- Top Police Forces by Recorded Crime
- Crime Distribution by Offence Group
- Crime Distribution by Offence Subgroup
- Interactive Slicers for Financial Year, Quarter, Police Force, and Offence Group

---

Key Insights

The dashboard enables users to:

- Identify police forces with the highest reported crime volumes.
- Compare crime trends across multiple financial years.
- Detect offence groups responsible for the largest share of recorded crimes.
- Monitor quarterly changes in crime patterns.
- Explore crime data interactively using dashboard filters.
